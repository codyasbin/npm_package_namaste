# namaste-lib

A tiny library that returns a fixed greeting message.

## Install

```bash
npm install namaste-lib
```

## Usage

CommonJS:

```js
const { getMessage } = require("namaste-lib");

console.log(getMessage());
// namaste sansar ma codyasbin
```

ESM:

```js
import { getMessage } from "namaste-lib";

console.log(getMessage());
// namaste sansar ma codyasbin
```

## API

### getMessage()

Returns:

- `string` -> `"namaste sansar ma codyasbin"`
